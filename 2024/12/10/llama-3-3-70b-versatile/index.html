Groq API 免费使用教程

第一步：获取 API

  1. 访问 https://console.groq.com/
  2. 完成注册并申请 API -> googlemail
  3. 保存获得的 API 密钥

第二步：配置 NextChat

  1. 打开 https://app.nextchat.dev/
    https://github.com/ChatGPTNextWeb/ChatGPT-Next-Web
  2. 在左侧设置中填入：
    设置->模型服务商->Google
    设置->自定义接口
    模型(model) llama-3.3-70b-versatile
    • 接口地址：https://api.groq.com/openai/
    • API Key：填入之前获取的密钥
    • 自定义模型：llama-3.3-70b-versatile

  • 足够普通用户日常使用
